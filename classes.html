Not sure what to search for? Here are some recommended classes based on your profile!

csc207

Software Design

Introduction to large-scale software design and development concepts.

csc209

Software Tools and Systems Programming

Software techniques in a Unix-style environment, using scripting languages and a machine-oriented programming language (typically C).

What goes on in the operating system when programs are executed.

Core topics:

  • creating and using software tools
  • pipes and filters
  • file processing
  • shell programming
  • processes
  • system calls
  • signals
  • basic network programming

csc236

Introduction to the Theory of Computation

The application of logic and proof techniques to Computer Science.

Mathematical induction; correctness proofs for iterative and recursive algorithms; recurrence equations and their solutions; introduction to automata and formal languages.

This course assumes university-level experience with proof techniques and algorithmic complexity as provided by CSC165H1. Very strong students who already have this experience (e.g. successful completion of MAT157Y1) may consult the undergraduate office about proceeding directly into CSC236H1.

csc343

Introduction to Databases

Introduction to database management systems including:

  • relational data models and algebra
  • querying and updating databases
  • application programming with SQL
  • database design
  • transaction management

Students will also learn about XML and using XQuery to query XML data.

csc411

Machine Learning and Data Mining

An introduction to methods for automated learning of relationships on the basis of empirical data.

Classification and regression using nearest neighbour methods, decision trees, linear models, and neural networks.

Studying clustering algorithms and problems of overfitting and of assessing accuracy.

Problems with handling large databases with an emphasis on statistical approaches.

csc443

Database System Technology

Implementation of database management systems. Storage management, indexing, query processing, concurrency control, transaction management.

Database systems on parallel and distributed architectures.

Modern database applications: data mining, data warehousing, OLAP, data on the web.

Object-oriented and object-relational databases.
